**4258MP3 Under-Shelf Clock Radio with CD/MP3 Player**

**Product Overview**: 
The 4258MP3 is designed for under-shelf installation, combining a clock radio with a CD and MP3 player for versatile audio playback.

### **User Instructions**:
- **Snooze/CD/Radio Off Button**: Activates snooze or turns off the CD/radio.
- **CD/Radio On Button**: Powers on the CD/radio.
- **Open/Close Button**: Opens and closes the CD tray.
- **Alarm Set Button**: Configures the alarm.
- **Clock Set Button**: Adjusts the time settings.
- **Hour/Minute Set Buttons**: Sets hours and minutes respectively.
- **Sleep Set Button**: Configures the auto-off timer.
- **Play/Pause Button**: Controls playback.
- **Stop Button**: Halts playback.
- **Skip Buttons**: Navigate through tracks.

**Safety Warnings**:
- **Electrical Shock Hazard**: Do not open the unit; there are no user-serviceable parts inside. For maintenance, contact qualified professionals.
  
### **Functionality and Operation**:
- **Power Supply**: Operates on 230V AC or 9V DC.
- **Clock Setup**: Power on and use the clock button to set the time.
- **Radio Operation**: Use the function switch to select AM or FM; adjust volume and tuning as necessary.
  
**CD/MP3 Playback**:
- To play CDs/MP3s, insert media and use the play/pause button as needed. Use the skip buttons for track navigation.
  
**Programmed Play**: 
- Enables the selection of tracks for customized playback. Follow instructions to set and program desired tracks for playback.

### **Alarm Setup**:
- Set alarms using the designated buttons to choose between buzzer or radio. The unit offers a standard snooze feature.
