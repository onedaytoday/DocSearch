**Introduction:**  
This manual provides details on the Touch 1517L and 1717L touchmonitors, emphasizing their performance features and functionalities. These monitors utilize an active thin-film transistor (TFT) LCD for high-quality display, enhanced by an LED backlight that reduces environmental impact compared to traditional CCFL backlighting. Additional capabilities include on-screen display (OSD) functionality and optional magnetic stripe reader (MSR) support.

**Unpacking the Touchmonitor:**  
Upon opening the package, ensure that the following items are included:
- Touchmonitor with protective covering
- TouchTools CD
- VGA cable
- USB cable
- AC-DC power adapter (North American model)

**Connector Interfaces:**  
The touchmonitor features a VGA input for video and a USB input for touch functionality. Ensure all connections are secured properly for optimal performance. The power can be turned on by pressing the designated button.

**Software Installation:**  
To use the touchmonitor with various operating systems like Windows XP, the setup CD must be utilized. Additional information on operating systems compatibility can be found on the official website at www.elotouch.com.

**Mounting Instructions:**  
For mounting, the OSD text can be rotated for easier viewing. The rear includes a VESA mount with four M4 screw holes, accommodating standard mounting practices. Ensure to follow the VESA coding: VESA MIS-B, C.

**Operational Guidelines:**  
To power on or off the touchmonitor, use the designated switch. The touchmonitor will indicate its status through LED signals, with specific meanings for SLEEP and OFF modes. Touch functionality is factory-calibrated, and the user is advised to refer to the OSD for configuration related to input settings and performance adjustments.

**OSD Functions:**  
The on-screen display (OSD) provides a graphical interface for adjusting brightness, clock phase, and scaling. Users can change the OSD settings including language options and timeout durations. Factory settings can be restored if needed.

**Technical Support:**  
For any technical issues, consult the OSD for brightness adjustments or reconnect the DC power supply if the touchscreen fails to respond. Further assistance can be sought via the official support channels.

**Cleaning and Maintenance:**  
To maintain the touchmonitor, disconnect from power and clean it with a soft cloth. Avoid using any abrasive materials or harmful cleaning solutions.
