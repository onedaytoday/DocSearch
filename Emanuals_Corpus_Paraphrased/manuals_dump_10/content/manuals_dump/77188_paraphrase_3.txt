### User Guide Overview

#### Basic Functions

1. **Power On/Off**
   - To power on, press the red "Power" button. Use the function selector to choose between RADIO, MP3/USB, or OFF.
   
2. **Volume Control**
   - Adjust the volume using the MIN/MAX control.

3. **Listening Options**
   - For FM/AM reception, select the respective function and fine-tune with TUNING and VOLUME controls.
   - To stop the CD/MP3 playback, switch functions to OFF.

#### CD and MP3 Playback

1. **CD and MP3 Functions**
   - Insert CDs or connect USB devices. The display will show status updates.
   - Start playback with the PLAY/PAUSE button. If playback is stopped, the display will indicate the current track.
   - Use SKIP- to return to previous tracks and SKIP+ to advance to the next tracks.

2. **Repeat and Program Playback**
   - To repeat a track, press REPEAT once; for all tracks, press it multiple times. 
   - Use the PROGRAM function to set a specific playback order.

#### USB Playback

- For USB playback, ensure the unit is set to USB mode. Similar controls (SKIP-, SKIP+, PLAY/PAUSE) apply as with CD playback.
- Status updates will show on the display during USB operation.

#### Cleaning and Maintenance

1. **CD Care**
   - Always handle CDs by the edges to avoid fingerprints.
   - Clean with a soft cloth, wiping from the center to the edge.
