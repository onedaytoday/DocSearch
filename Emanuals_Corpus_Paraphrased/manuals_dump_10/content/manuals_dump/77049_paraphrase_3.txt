### Introduction

This document provides an overview of the performance and features of the 1940L widescreen touchmonitor, which utilizes an active thin-film-transistor LCD and offers high-definition graphics. The LED backlight is mercury-free, enhancing efficiency compared to traditional CCFL-backlit screens. Follow the safety precautions outlined in this document to ensure correct usage.

### Package Contents

The 1940L SW601787 package includes:
- Touchmonitor
- Protective sheet
- TouchTools CD
- DVI/VGA cables
- USB interface
- Remote OSD connector

### Setup Instructions

Connect the touchmonitor using DVI or VGA ports, ensuring screws are tightened for optimal performance. Connect the USB for functionality and plug in the 12V DC power supply. The monitor will power on, displaying the initial screen.

### Software Installation

The touchmonitor is compatible with XP and WEPOS operating systems. For installation, refer to the TouchTools CD, or visit www.elotouch.com for additional support and software options.

### Mounting Information

The touchmonitor features a rear VESA mount, compatible with VESA MIS-D, C specifications. Use M4 screws to secure the monitor during installation.

### Operating Instructions

To power on or off the touchmonitor, use the LED indicators and the Remote OSD. The monitor includes factory calibration, and energy-saving features can help reduce power consumption. The touchscreen responds to interactions and operates in alignment with HID standards for enhanced usability.

### On-Screen Display (OSD)

The OSD allows users to adjust settings such as brightness, clock phase, and aspect ratio. The interface provides multiple languages and can be navigated to select different input sources. Adjustments can be made to improve image quality based on user preferences.

### Troubleshooting

For issues with functionality, ensure all connections are secure. Users can refer to technical assistance resources at www.elotouch.com or contact support for additional help. Avoid using harsh chemicals when cleaning the touchscreen—opt for mildly damp cloths only.
